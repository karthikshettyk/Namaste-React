7)What is the difference between react.develeopment.js files and react.production.js files via CDN?
Ans:React.development.js files are used to write the code locally and debugging
    ex:
     <script src="https://unpkg.com/react@18/umd/react.development.js"></script>

     <script src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
